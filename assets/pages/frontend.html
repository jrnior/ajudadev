HTML e criação de sites

O HTML (HyperText Markup Language) é a espinha dorsal de toda a web e o principal componente para a criação de sites, sendo a linguagem que define a estrutura e o conteúdo de uma página. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação que usa "tags" (etiquetas) para envolver e descrever diferentes partes do conteúdo, como títulos, parágrafos, imagens e links.
